# Node.js 04 — Parse Headers

> **Pattern:** HTTP parsing  
> **Function:** `parseHeaders`  
> **Exercise:** `src/exercises/nodejs-backend/01-http-fundamentals/04-parse-headers/`

## Problem

Parse `Key: value` lines. Lowercase keys. Trim values.

## Hint

Split on newlines, first colon separates key/value.

## Run locally

```bash
npm run node:04
```
