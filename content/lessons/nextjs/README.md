# Next.js — full course (framework only)

**10 course lessons** — no exercises. Focus on what Next.js adds on top of React: routing, RSC, API, caching, Server Actions, middleware, config, deploy.

| # | Lesson | Topic |
|---|--------|--------|
| 01 | [Overview & tooling](./01-overview-and-tooling.md) | Why Next, CLI, project layout |
| 02 | [App Router & routing](./02-app-router-routing.md) | `app/`, segments, dynamic routes |
| 03 | [Server vs Client](./03-server-client-components.md) | RSC, `"use client"`, boundaries |
| 04 | [Route Handlers (API)](./04-route-handlers-api.md) | REST in `app/api`, this repo's `/api/*` |
| 05 | [Data fetching & cache](./05-data-fetching-caching.md) | `fetch`, revalidate, tags |
| 06 | [Layouts & special files](./06-layouts-loading-errors.md) | `layout`, `loading`, `error`, `not-found` |
| 07 | [Server Actions](./07-server-actions-forms.md) | Forms, mutations, `useActionState` |
| 08 | [Middleware & env](./08-middleware-auth-env.md) | Edge, auth patterns, secrets |
| 09 | [Config & deploy](./09-config-build-deploy.md) | `next.config`, build, Vercel |
| 10 | [Interview Q&A](./10-interview-questions.md) | Recap for technical interviews |

In the app: `/lessons/nextjs/01-overview-and-tooling`, etc.

This repo also has **hands-on labs** under tracks easy / medium / hard (Redux + API exercises).
