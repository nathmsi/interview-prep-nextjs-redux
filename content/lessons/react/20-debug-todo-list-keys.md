# React — Debug: List Keys

> **Exercise:** `content/exercises/react/03-debug-todo-list-keys/`  
> **Run:** `npm run react:03`

## Problem

Editable list rows must keep their **input value** when items are **reordered**.

## Hint

`key={index}` reuses the wrong DOM nodes. Use a **stable id**.

## Interview angle

« Why not index as key? » — show this exact bug live.
