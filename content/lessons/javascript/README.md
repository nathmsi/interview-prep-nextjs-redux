# JavaScript — interview prep

Three **course-only** lessons (no exercises):

| Lesson | Level | Focus |
|--------|-------|--------|
| [01-basic](./01-basic-interview.md) | Basic | Types, scope, equality, arrays, functions |
| [02-medium](./02-medium-interview.md) | Medium | Closures, `this`, prototypes, async, event loop |
| [03-hard](./03-hard-interview.md) | Hard | Memory, microtasks, advanced patterns, TS tie-in |

Examples use **JavaScript** with **TypeScript** snippets where typing clarifies the answer (same as real interviews: you can explain in JS and show TS for APIs).

Open in the app: `/lessons/javascript/01-basic-interview`, etc.
